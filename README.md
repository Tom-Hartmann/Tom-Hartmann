# Tom-Hartmann's Discord Bot

A versatile Discord bot built to enhance server interaction and management.

## Features

- **Auto Moderation:** Enforce rules consistently and fairly.
- **Welcome & Goodbye Messages:** Customizable messages for members joining and leaving with custom images.
- **Logging:** Monitor important events in your server.
- **Steamhourboost:** Setup multiple accounts to boost hours in games.
- **Anti-Link:** Delete links when sent.
- **Auto-Role:** When joining grant people roles.

## Installation & Setup

1. **Clone the repository**
   \```
   git clone https://github.com/Tom-Hartmann/Tom-Hartmann.git
   \```

2. **Navigate to the directory**
   \```
   cd Tom-Hartmann
   \```

3. **Install dependencies**
   \```
   npm install
   \```

4. **Configuration**

   - Rename `config.example.json` to `config.json`.
   - Fill in the required fields, such as your bot token.

5. **Start the bot**
   \```
   node . or node index.js
   \```

## Usage

for `<prefix>` use your prefix you wrote in the config at `DEFAULT_PREFIX`

- `<prefix>help`: Display a list of available commands.
- `<prefix>welcome <message>`: Set a custom welcome message.
- `<prefix>goodbye <message>`: Set a custom goodbye message.
- or check out the slash(/) commands available
